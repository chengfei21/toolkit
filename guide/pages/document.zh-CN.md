# 快速上手

## 安装

安装工具库：

```bash
$ yarn add @xylink/toolkit
```

## 更新

```bash
$ yarn upgrade @xylink/toolkit
```

## 使用

```ts
import { sotre, fscreen, ... } from @xylink/toolkit;
```
